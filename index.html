Mozilla Firefox

Mozilla Firefox is a free open-source web browser developed by the Mozilla Foundation. Here are some facts about Firefox:

  • Firefox was created in 2002.
  • It was initially released in November 2004.
  • It is regarded as the second-most popular browser choice by usage share.
  • Firefox is available in 90 languages.

Mozilla Firefox is known for it's speed, security and customisability. However it has been criticised by some for it's memory usage.

You can learn more about Firefox by visiting the Mozilla Manifesto page.
